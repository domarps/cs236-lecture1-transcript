# CS236 Lecture Transcript Search

Static searchable transcript for Stanford CS236 Lecture 1, "Introduction."

Open `index.html` locally, or host this repository with GitHub Pages from the
repository root. Search results include clickable timestamps that jump to the
matching point in the YouTube lecture.

## Files

- `index.html` - GitHub Pages entry point with the searchable transcript app.
- `lecture1_search.html` - Same searchable page under its original generated name.
- `lecture1_transcript.md` - Timestamped Markdown transcript.
- `data/lecture1_transcript.json` - Structured timestamped transcript data.
